# Build HTML Mockup

## 目的

機能実装を行わず、設計済み画面を静的HTMLモックとして作成する。

## 手順

1. `design-system/` と `03_design/screen-design/` を読む。
2. `05_mockup/pages/` に画面ID付きHTMLを作成する。
3. CSS/JSは最小限にする。
4. DB接続、API、認証、本実装は行わない。
5. 完了後、`03_design/screen-function-matrix.md` のモック作成欄を更新する。

## 完了条件

- 対象画面のHTMLモックが存在する。
- 画面機能一覧表が更新されている。
