# Design Screens and Functions

## 目的

要件定義から画面設計、機能設計、DB設計、画面機能一覧表を作成する。

## 手順

1. `02_requirements/requirements-definition.md` を読む。
2. 画面一覧と画面遷移を作成する。
3. 画面ごとの表示項目、操作、権限、バリデーションを設計する。
4. 機能一覧と機能詳細を作成する。
5. DB設計、ER、テーブル定義、migration方針を作成する。
6. `03_design/screen-function-matrix.md` に要件ID、画面ID、機能IDを紐づける。

## 完了条件

- 画面、機能、DB、テスト対象が追跡可能になっている。
