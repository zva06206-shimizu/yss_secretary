# ROI Template

## 目的

営業提案書で、投資対効果を現実的に説明するためのテンプレート。

## ROI基本式

```txt
年間効果額 = 月間削減時間 × 時間単価 × 12か月 + その他効果
投資回収期間 = 初期費用 ÷ 月間効果額
```

## 注意

- 効果額は過大に見せない。
- 未確認数字は仮説と明記する。
- 顧客から聞いた数値と、当社試算を分ける。
- 定量化できない効果は、無理に金額化しない。
