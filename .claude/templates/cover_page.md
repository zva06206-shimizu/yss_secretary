# Cover Page Template

```html
<section class="cover">
  <div class="cover-title">{{proposal_title}}</div>
  <div class="cover-subtitle">{{client_name}} 御中</div>
  <div class="cover-meta">
    <p>提出日：{{submitted_date}}</p>
    <p>作成：{{company_name}}</p>
    <p>版数：{{version}}</p>
  </div>
</section>
```

## 記載ルール

- 提案名は具体的にする。
- 「DX提案書」のような抽象名だけにしない。
- 顧客名、提出日、作成会社、版数を必ず入れる。
- 表紙は1ページで独立させる。

## 良いタイトル例

- 案件進捗と見積業務の見える化システムご提案書
- Larkを活用した営業管理・業務改善システムご提案書
- AI活用による問い合わせ対応・ナレッジ共有改善ご提案書
