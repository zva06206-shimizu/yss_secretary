# sankaku（副業マッチング「サンカク」応募管理）

副業マッチングサイト **サンカク** 経由の**応募案件のパイプライン**をここで管理する。

## 運用ルール

1. **応募したら**ここに登録する：[application-tracker.md](application-tracker.md) に1行追加し、必要なら `work/sankaku/{案件}/` にフォルダを作る。
2. **選考が進む**間（応募済 → 面談 → 契約オファー）はここで管理する。
3. **採用が確定したら**、その案件フォルダを `work/{案件}/` 直下へ**昇格（git mv）**し、正式案件として運用する。application-tracker の状態を「採用→昇格済」にする。
4. **見送り・不成立**は状態を更新して履歴として残す（消さない）。
5. サンカクの**手数料は採用側（企業）負担＝YSS側は費用負担なし**（契約額が満額入金）。直接契約に移す金銭メリットは薄いため、直接化は必要時のみ検討。

## メモ

- パーツ精工は応募〜面談が先行したため `work/parts-seiko/` に先行作成済み。**配置（sankaku配下に下げるか top-level 維持か）は2次面談の結果で判断**。
- 状態区分：`応募済` / `書類通過` / `面談中` / `契約オファー` / `採用(→昇格)` / `見送り`
