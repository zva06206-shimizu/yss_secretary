## 4-2. 原価・粗利・「収入のはしご」（重要）

### 業界の一般目安（FLR）
- **F 原価（食材）30%前後**／L 人件費30%前後／**R 家賃10%以下が理想**。FL60%以下・FLR70%以下が健全。
- 売価設定は「原価の3倍＝原価率33%」が基本。**ドリンク・酒は原価20〜25%**で粗利が高い→混ぜると総合原価率を下げられる。

### この店の特殊構造
- **L（人件費）≒0**（本人調理）だが、**R（家賃）が重い**：家賃17.6万÷売上45万＝**39%**（理想の約4倍）。
- 「本人がタダ働きして重い家賃を相殺し黒字化」する構造。だから本人調理は外せない。

### 売上別の粗利と取り分（原価率35%・固定費26.6万）

| 月売上 | 粗利 | 営業利益＝本人取り分 | 位置づけ |
|---:|---:|---:|---|
| 41 | 26.6 | 0 | 損益分岐 |
| 45 | 29.3 | +2.6 | 店は回るが取り分ほぼ無し |
| 50 | 32.5 | +5.9 | |
| 60 | 39.0 | +12.4 | |
| 65〜70 | 42〜45 | **+15〜19** | **本人が生活費を取れる＝自立ライン** |

> **45万は「店が潰れないライン」であって「彼女が暮らせるライン」ではない。** 目標は2段階：**①42万で潰さない → ②65〜70万で自立**（＝Yutakaの生活費負担減という当初目的を達成）。

### 粗利を上げる打ち手（値下げより効く）
1. ドリンク・酒の比率を上げる（原価20%台）→総合原価率30%以下に。
2. カラオケ課金（原価ほぼ0＝丸々粗利）。
3. セットにドリンクを付けて客単価UP（SET B 1500→ドリンク付2000等）。
4. 週替わりは良いが仕込みロスがFを押し上げる→食材を使い回せる構成でFを安定。

## 5. 反証メモ（ここを詰めないと机上の空論）

1. **【最重要・許認可】持ち込み調理の壁**：飲食店営業許可は"調理する場所"に紐づく。**自宅で作って店に持ち込むのは保健所NGの可能性大**。→ 解は「**店の厨房でパートナー自身が仕込み・調理**（料理人は雇わない）」。これなら本人調理＝コスト不要と、許可の整合を両立できる。明日TON＋保健所で確認。
2. **イベント25万を日常に外挿しない**：イベントは数千人規模の催事。実店舗の日販はコミュニティの来店頻度依存。**需要の存在証明としては有効、日販予測には使えない**。土台はrecurringで見る。
3. **ワンオペの上限**：本人が調理＋ホール＋会計は capacity に限界。**月45万前後が一人の現実的上限**。それ以上に伸びたら増員＝人件費復活。「まず45万で黒字、伸びたら投資」。
4. **原価率は未確定**：イベント構成比表は原価未入力（粗利＝売上のまま）。**実際の食材原価を入れて真の粗利率を出す**。揚げ物（martabak/ayam/otak-otak）は油・廃棄ロスに注意。
5. **【隠れコスト】送迎とYSSの競合**：パートナーが自走できず、自宅(大治町)〜店(高畑)をYutakaが送り迎えする場合、**YSS本体（収益の柱・33万/月＋スポット）の顧客訪問と時間・車（セレナ）が競合**する。"人件費ゼロ"は、Yutakaの時間と車を使うなら見かけ上の話。**YSSの稼働を1日でも潰せば、店の薄利を軽く上回る損失**。
   - 原則：**YSSの客先訪問（売上に直結）＞ 店の送迎**。送迎前提で店を設計しない。
   - 対策候補：①パートナーの通勤を独立させる（免許取得・自転車/原付・公共交通の可否を確認。定住者は免許取得可）。②**営業を夜＋週末中心**にし、YSS訪問が多い平日日中と時間帯を分離（夜営業はもともと業態に合う）。③仕込み搬入の時間帯もYSS稼働と重ねない。
   - 車両費の按分も複雑化（YSS事業按分とは別の私用/別事業走行が増える）。家事按分細則の走行記録で分けること。
