# 03_documents — 契約書・議事録・提出文書

## 契約書（業務委託契約・YSSとして締結）

- 原本ファイルをこのフォルダに置く。命名例：`業務委託契約書_YSS-カヨウ_YYYYMMDD.pdf`
- 置いたら、要点を `契約サマリ.md` に整理する（下記項目）。
- **契約金額など秘匿性の高い数値は、公開追跡ファイル側では要約にとどめる**（生値はサマリに残さず、必要なら git 管理外へ）。

### 契約サマリに残す項目
- 契約当事者（YSS ／ 株式会社カヨウコーポレーション）
- 契約日・契約期間
- 業務範囲（スコープ）＝ **前回どこまでを請けたか**
- 報酬・支払条件（要約）
- 成果物・納品条件
- 秘密保持・再委託・中途解約の条項で効くもの
- **前回スコープ外だった＝今回の再提案の種になる領域**
