# カヨウコーポレーション（案件ワークスペース）

株式会社カヨウコーポレーション（静岡県藤枝市）の案件フォルダ。
**業務委託契約を締結して対応した案件**の実物（契約・提案・対応記録・成果物・受領資料）をここに残す。

長期文脈の正本（会社理解・キーパーソン・振り返り分析）は
[data/memory/clients/kayo-corp.md](../../data/memory/clients/kayo-corp.md)。

## この案件のポイント（詳細は正本参照）

- 紹介＝**顧問会計士 佐藤信一氏**（カヨウの会計を継続して見ている＝内部スポンサー候補）。
- 代表＝**大久保良平氏（2代目社長）**。
- テーマ＝**DX・業務効率化**。
- 位置づけ＝**過去に業務委託契約を結び対応した案件の振り返り**。

## フォルダ構成

```txt
00_context/    会社・案件の背景、キーパーソン、契約前提
01_business/   業務整理・現状分析
02_proposal/   提案書・見積
03_documents/  契約書（業務委託契約）・議事録・顧客提出文書
04_system/     システム関連（設計・構成・成果物）
05_operation/  運用・保守・改善
06_data/       受領資料・データ（機密は要約 or git管理外）
99_logs/       打合せメモ・作業ログ
```

## 記録ルール（CLAUDE.md 準拠）

- 事実・仮説・未確認を分ける。
- 秘密情報・個人情報・契約金額の生値は要約にとどめるか git 管理外へ。
- 案件をまたぐ学びは `data/` に戻す。
