# 5号館 フロア高 実測（2026-06-23・レーザー計測）

調査者：清水（YSS）。用途＝縦系配線の距離試算（home-run/スター化の可否、光上り要否）。

## 実測値（各階フロア高 m）

| 階 | フロア高(m) | NW機器 |
|---|---|---|
| 2F | 2.657 | あり（教員/生徒AP・配電盤PoE） |
| 3F | 2.612 | あり |
| 4F | 2.634 | あり |
| 5F | 2.602 | **なし（ネットワーク機器・配線なし＝NWスコープ外）** |

※1F高は未計測（コア設置階）。NW対象は **1F〜4F**。5Fは建屋としては存在するがネットワーク対象外。

## home-run（スター化）距離試算

現状＝各階PoE(BS-GS2008P)を1F→2F→3F→4Fでカスケード。改善案＝1Fコアから各階IDFへ home-run。

- 1Fコアから最上NW階(4F)までの**縦距離 ≒ 1F高 + 2F(2.657) + 3F(2.612) ≒ 約7.9m**（1F≒2.6m想定）。
- これにriser取り回し＋各階フロア内の水平ラン（最遠AP=54系まで）を加算。
- **試算：縦8m＋水平40〜60m＋余長 ≦ 約100m** → **銅(Cat5e/6)のhome-runで100m以内に収まる見込み＝光上り不要**。
- ∴ **5号館はカスケード→1Fコアからの銅スター化が物理的に成立**（6号館8Fのような光要否判断は不要）。最終確定は各階の水平最遠ランの実測で詰める。

→ N-02提案：**現状の数珠つなぎ（単一経路・SPOF）を、1Fコア＋管理SWのhome-runスターに巻き直し可能**、という根拠データ。空き管(riser)の有無と合わせて可否を確定する。
