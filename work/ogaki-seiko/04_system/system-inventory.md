# 大垣精工 社内システム一覧（現状）

出典：先方提供『社内システム簡易一覧.xlsx』（同フォルダ）をテキスト化。

## 会社・事業区分（重要）

- **OSK ＝ 大垣精工株式会社（金型事業）**
- **SHT ＝ 株式会社セイコーハイテック（量産事業）**
- 法人としては別会社だが、**事業別の位置づけで実際は同じ会社内**という扱い。
- このため生産管理・会計システムがOSK系／SHT系で二重に存在し、情報統合が課題の核心。
- ドメイン：`ogakiseiko.co.jp`（本体）とは別に `oskpre.com` をGoogle Workspace（AI活用）用に運用。

## 所見（クロエ・反証メモ）

- **二重システムが課題の本丸**：OSK＝OSK-DB＋Dr.原価Pro＋TKC、SHT＝TECHS-BK＋ICS。生産管理も会計も二系統。提案書の「システム統一ではなく情報統合を優先」は妥当な落とし所。
- **既に持っているが活かせていない資産**が多い：TECHSの在庫・製造履歴機能、楽楽精算（電子保存のみ）。新規導入より**既存の活用度を上げる**方が即効性・低コストで成果を見せやすい＝9月30万転換のROI材料に向く。
- **共有アカウント98・共有メール**は退職異動時の権限管理リスク（提案書の組織課題と一致）。※ISMS（情報セキュリティの管理体制）はNTT担当だが、**アカウント／アクセス権管理（Active Directory）はYSS提案範囲**（4/30ロードマップに記載）として有効。両者を切り分けて提案する。
