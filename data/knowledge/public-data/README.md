# Public Data Knowledge Base

## 目的

このディレクトリは、DX提案、経営分析、補助金資料、研修資料、業界分析に使うための公的情報源を整理する場所である。

対象は、日本国内および海外の白書、年次報告、政策レポート、統計ポータル、国際機関レポートである。

## 基本方針

- 公式・準公式情報を優先する。
- 民間記事や二次情報は、一次情報に当たるための補助として扱う。
- 海外では White Paper という名称に限定しない。
- Policy Paper、Green Paper、Strategy Paper、Annual Report、Flagship Report、Outlook、Survey、Statistical Bulletin、Government Report、Official Statistics、Publications、Research and Statistics、Working Paper、Consultation Paper も対象にする。
- 提案書に使う場合は、URLだけでなく「何の根拠として使うか」を明記する。
