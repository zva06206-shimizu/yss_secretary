# Knowledge

## 目的

このディレクトリは、Claude Codeが経営相談、業務改善、提案書作成、システム設計、AI活用、セキュリティ、UI設計に関する公式知識を、リポジトリ内で再利用するための共通知識ベースである。

## 中核ファイル

| File | Purpose |
|---|---|
| `source-registry.yml` | 参照すべき公式・標準・公的機関ソース一覧 |
| `public-data/` | 白書、公的統計、政策レポート、国際機関レポート |
| `distilled-rules/` | 実務ルールへ変換した知識 |
| `summaries/` | 各ソースの要約 |
| `standards/` | 標準、規格、公式ガイドライン |

## 運用

1. 新しい情報源を追加する場合は `source-registry.yml` に登録する。
2. 実務上のルールに変換できるものは `distilled-rules/` に反映する。
3. 案件で参照した場合は、`work/{project}/99_logs/decision-log.md` または成果物内に出典を記録する。
4. 原文全文保存はライセンス上問題ない場合のみ行う。
5. 公的白書や統計資料は `public-data/` で管理する。

## 原則

URLを登録するだけで完了ではない。各ソースは、工程、成果物、チェックリスト、品質ゲートへ紐づいた状態を完了とする。
