# Cost Structure（固定費・損益分岐の素地）

## 目的

「安定して案件を取れる状態」の**最低月商ライン**を出すための、固定費・生活費の把握。
個人事業主のため、(A)事業の経費 と (B)生活費（事業所得から賄う・経費外）を分けて管理する。
※財務数値は機密。Privateリポジトリ前提で最小限のみ記録。

## 構造メモ（事実 / 仮説 / 要確認）

- **【良い変化・事実】** 2026年6月から家賃120千円がゼロに。生活防衛ラインが大きく下がった。
- **【リスク・事実】** 仕事場按分6万円/月が「入金不足で払えていない」。キャッシュが最低限の固定費すら回せていないサイン。売上の入金タイミング（現金主義）と支払いのズレが効いている可能性。
- **【要確認・税務】** パートナー（事実婚）への仕事場家賃6万円が経費に算入できるかは税理士確認が必要。所得税法56条（生計を一にする親族への対価は原則経費不算入）の適用有無は、法律婚か否か・生計同一かで変わる。**断定せず専門家に確認**。
- **【未確認】** (A)(B)の未確認額が埋まれば「最低必要月商ライン ＝ 事業経費 ＋ 生活費 ＋ 税・国保・年金」が試算でき、「安定」の数値目標が引ける。
