# Company Memory

## 目的

会社の事業、組織、従業員、経営状況、顧客、課題、KPIをリポジトリ内に蓄積し、Claude Codeが経営者の右腕として一貫した提案・判断・設計をできるようにする。

## 管理する情報

- 会社概要
- 事業内容
- 収益構造
- 商品・サービス
- 組織・従業員
- 役割分担
- 経営KPI
- 経営課題
- 顧客・案件状況
- 提案方針
- 採用・教育・定着状況
- 今後の重点施策

## 注意

- 事実、仮説、未確認を分ける。
- 個人情報は必要最小限にする。
- 従業員情報は業務上必要な範囲に限定する。
- 財務情報や人事情報は機密扱いとする。
- 古い情報は更新日を明記する。
