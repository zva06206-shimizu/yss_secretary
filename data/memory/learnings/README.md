# Learnings（横断的な学び）

案件・会議・数字分析から得た、繰り返し使える学びを残す。Skill `13-record-executive-context` / `14-analyze-numbers-and-act` などが学びを戻す先。

## ルール

- 1学び = 1ファイル、または論点ごとに追記。命名は `{topic}.md`。
- 事実・仮説・未確認を分ける。
- 「次回どう活かすか」を必ず書く。書けない学びは残さない。
- 判断軸に昇格すべき学びは `data/memory/decision-principles.md` へ移す。

## フォーマット

```md
# {学びのタイトル}

- 起点（案件/会議/数字分析）：
- 事実：
- そこから言える仮説：
- 次回への活かし方：
- 更新日：
```
