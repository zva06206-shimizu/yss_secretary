# Payment Tracker（入金・支払・引落 消込）

毎月の**入金（売上）・支払・引落・税金・年金**を予定として持ち、「払ったか／引き落とされたか／入金されたか」を毎月チェックして消し込む。
資金繰りの全体像は [work/yss-cashflow](../../../work/yss-cashflow/05_operation/cashflow-2026H2.md)、固定費の内訳は [cost-structure.md](../company/cost-structure.md)。

- 区分：`入金` / `支払` / `引落` / `税金`
- 状態：`⬜未` / `✅済` / `—該当なし`
- 金額は目安。確定したら実額に直す。**「要確認」は通帳・カードアプリで確定して埋める**。
- 機密度の高い金額は要約にとどめる。

最終更新: 2026-06-22

## 使い方

1. 毎月初に、マスタ（A）からその月の消込表（B）を作る。
2. 入金・引落を確認したら状態を `✅済` にし、実績日・実額を記入。
3. 月末に未消込（⬜未）が残っていないか確認。ズレは資金繰り（cashflow）に反映。
